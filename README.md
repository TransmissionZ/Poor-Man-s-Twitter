# Running Poor Man's Twitter
```
1) python manage.py runserver (should be on port 8000)
```
```
2) Open new terminal and cd poormans_vue
```
```
3) npm run serve (should be on port 8080)
```
```
4) open http://localhost:8080 on your browser
```
